addFragment

class addFragment : Fragment

Fragmento para agregar un nuevo piloto.

Este fragmento proporciona una interfaz para agregar un nuevo piloto con información como nombre, número, puntos, etc. Se comunica con el controlador de la aplicación para realizar operaciones relacionadas con la base de datos y la lógica de la aplicación.
