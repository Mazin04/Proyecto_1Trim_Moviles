LoginRegisterActivity

class LoginRegisterActivity : AppCompatActivity

Actividad principal para el inicio de sesión y registro de usuarios.

Esta actividad permite a los usuarios iniciar sesión o registrarse en la aplicación. Proporciona funcionalidades para cambiar entre los modos de inicio de sesión y registro, además de validar y procesar la información proporcionada por los usuarios.
