modifyFragment

class modifyFragment : Fragment

Fragmento para modificar la información de un piloto existente.

Este fragmento proporciona una interfaz para seleccionar y modificar la información de un piloto existente en la base de datos. Se comunica con el controlador de la aplicación para realizar operaciones relacionadas con la base de datos y la lógica de la aplicación.
