class deleteFragment : Fragment

Fragmento para eliminar un piloto existente.

Este fragmento proporciona una interfaz para seleccionar y eliminar un piloto existente de la base de datos. Se comunica con el controlador de la aplicación para realizar operaciones relacionadas con la base de datos y la lógica de la aplicación.
